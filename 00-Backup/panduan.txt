Untuk proyek bot game berbasis data JSON dan pathfinding (pencarian rute) seperti ini, ada dua pustaka (library) yang jika diinstal akan berdampak sangat besar dalam memotong jumlah baris kode dan mencegah eror tipe data secara signifikan:
1. Pydantic (Untuk Validasi & Pemetaan Data JSON)
Proyek Anda menerima banyak sekali payload JSON dari server yang strukturnya bercabang-cabang dan dinamis. Selama ini, Anda harus menulis banyak validasi manual untuk mencegah crash, seperti:
isinstance(log_data, dict)
view.get("self", {}) or {}
.get("maxHp") or .get("max_hp", 100)
Jika menggunakan Pydantic, Anda cukup mendefinisikan struktur data (skema) dalam bentuk Class Python. Pydantic akan otomatis menyaring, memvalidasi tipe data, dan mengubah JSON mentah menjadi objek Python yang aman digunakan.

2. NetworkX (Untuk Navigasi & Pencarian Rute / Pathfinding)
Di dalam movement_strategy.py dan ruin_exploration_strategy.py, Anda menulis algoritma pencarian rute terpendek secara manual menggunakan metode BFS (Breadth-First Search) yang memakan puluhan baris kode logika antrean (queue), pelacakan node yang dikunjungi (visited), dan kalkulasi bobot rute.
Jika menggunakan NetworkX (pustaka khusus pengolah graf dan jaringan), semua logika BFS/Dijkstra manual tersebut bisa diganti hanya dengan beberapa baris fungsi bawaan yang jauh lebih cepat dan bebas bug.
