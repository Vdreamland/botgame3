/botgame2helpers/
- helpers/api_config.py
Mengatur: Parsing dinamis terhadap file .env berbasis indeks untuk mengekstrak daftar konfigurasi tiap-tiap agen. Menyediakan rute REST API, WebSocket URL, penanganan X-Version, dan skema alur Tiga-Dompet terpisah per instansi bot.

- game_constants.py
Mengatur: Waktu turn (30 detik), batas regenerasi EP, penalti flat cuaca (weatherMod), dan kepatuhan status set lengkap loadout (fullSet).

- actions_payload.py
Mengatur: Struktur JSON payload untuk dikirimkan lewat WebSocket, terbagi atas Aksi Ber-cooldown (EP-consuming) dan Aksi Instan (0 EP).

- items_spec.py
Mengatur: Tabel stat/jarak senjata (melee & ranged), pertahanan baju zirah (armor), efek item pemulihan (recovery), dan parameter acak reforge.

- entities.py
Mengatur: Stat monster (Wolf, Bear, Bandit) beserta loot drop-nya, lokasi spawn, dan tindakan agresif Guardian (tanpa efek Curse).

- quests_seasons.py
Mengatur: Tanggal Preseason 1, parameter 10 stepped quest (kurva Exp, Diminish, Linear), claim harian/mingguan (Rabu UTC), serta alokasi poin CROSS.

- errors_handler.py
Mengatur: Penanganan error REST, daftar kode pemutusan WebSocket (Close Codes 4xxx), deteksi limitasi IP (max 5), dan metode recovery.

- strategy_brain.py
Mengatur: Formula kalkulasi damage pertarungan, hierarki prioritas target AI, navigasi reruntuhan (ruins gauge), serta perekaman memori taktis regional agen.

- rungame.py
Mengatur: Inisialisasi thread/task asinkron secara simultan (paralel) sesuai dengan jumlah NUM_BOTS yang terdefinisi di .env. Menjalankan State Router dan WebSocket game loop yang independen untuk tiap-tiap instansi agen secara aman tanpa tabrakan data (thread-safe).

- .env
Mengatur: Jumlah bot aktif (NUM_BOTS), preferensi tipe room default (ROOM_PREFERENCE), serta kredensial nama dan API Key unik terindeks untuk masing-masing instansi agen (BOT1_NAME, BOT1_API_KEY, BOT2_NAME, dsb).
