/botgame3/
game_logs.py
Mengatur: Handler penulisan log terpusat menggunakan format Bahasa Inggris (English logging). Menjamin tampilan log aman secara asinkron (thread-safe/async-safe) agar tidak tumpang tindih saat melacak ratusan bot sekaligus. Menyediakan penanda warna visual/indeks bot untuk melacak transisi status koneksi: Handshake (hello), Antrean (queued), Pencocokan (assigned/joined), dan Penanganan Error.
rungame.py
Mengatur: Pengambilan berkas .env untuk inisialisasi paralel ratusan task bot dengan jeda bertahap (staggered launch) demi menghindari IP-ban. Menjalankan verifikasi versi, koneksi asinkron ke /ws/join, pengiriman payload jabat tangan hello dengan preferensi room (free atau paid), hingga transisi status berhasil masuk ke room arena. Semua pesan status dilempar ke game_logs.py untuk ditampilkan.
.env
Mengatur: Jumlah ratusan bot aktif (NUM_BOTS), pengaturan tipe room global (ROOM_PREFERENCE), serta rincian kredensial unik setiap bot secara terindeks untuk langsung masuk antrean.
requirements.txt
Mengatur: Pendataan dependensi pustaka Python minimalis wajib (requests, websockets, python-dotenv) beserta batas minimum versinya untuk menjamin stabilitas fungsionalitas asinkron game loop.

/botgame3/helpers/
helpers/__init__.py
Mengatur: Inisialisasi folder helpers sebagai paket Python modular untuk memudahkan impor bersih di seluruh bagian proyek.
helpers/api_config.py
Mengatur: Parsing dinamis terhadap file .env berbasis indeks untuk mengekstrak daftar konfigurasi tiap-tiap agen. Menyediakan rute REST API, WebSocket URL, penanganan X-Version, dan skema alur Tiga-Dompet terpisah per instansi bot.
helpers/game_constants.py
Mengatur: Waktu turn (30 detik), batas regenerasi EP, penalti flat cuaca (weatherMod), dan kepatuhan status set lengkap loadout (fullSet).
helpers/actions_payload.py
Mengatur: Struktur JSON payload untuk dikirimkan lewat WebSocket, terbagi atas Aksi Ber-cooldown (EP-consuming) dan Aksi Instan (0 EP).
helpers/items_spec.py
Mengatur: Tabel stat/jarak senjata (melee & ranged), pertahanan baju zirah (armor), efek item pemulihan (recovery), dan parameter acak reforge.
helpers/entities.py
Mengatur: Stat monster (Wolf, Bear, Bandit) beserta loot drop-nya, lokasi spawn, dan tindakan agresif Guardian (tanpa efek Curse).
helpers/quests_seasons.py
Mengatur: Tanggal Preseason 1, parameter 10 stepped quest (kurva Exp, Diminish, Linear), claim harian/mingguan (Rabu UTC), serta alokasi poin CROSS.
helpers/errors_handler.py
Mengatur: Penanganan error REST, daftar kode pemutusan WebSocket (Close Codes 4xxx), deteksi limitasi IP (max 5), dan metode recovery.
helpers/strategy_brain.py
Mengatur: Formula kalkulasi damage pertarungan, hierarki prioritas target AI, navigasi reruntuhan (ruins gauge), serta perekaman memori taktis regional agen.
helpers/state_router.py
Mengatur: Alur penentuan status kesiapan bot berbasis REST API (GET /accounts/me) sebelum membuka koneksi WebSocket.
helpers/game_connection.py
Mengatur: Siklus hidup koneksi WebSocket terpadu (/ws/join) mulai dari handshake, pemrosesan status antrean, hingga transisi aman ke soket permainan.
helpers/combat_controller.py
Mengatur: Pengendali asinkron in-game (AI Controller) yang mengevaluasi agent_view untuk memutuskan urutan aksi taktis saat bertarung di arena.
helpers/automation_scheduler.py
Mengatur: Penjadwalan otomatisasi tugas REST API di luar game seperti klaim quest harian/mingguan, reforge, marketplace, dan inbox notifikasi.
